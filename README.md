csv2Sqlite
==========

csv 2 sqlite. 

csv fileName will been DB Table Name.

you can edit dirPath and DBpath from __init__.py. and run __init__.py. 

Example
==========

        """example csv file:
            ID,Name,Age
            1,Wandy,12
            2,Ying,11"""
            
            DB.
            ID,Name,Age
            1,Wandy,12
            2,Ying,11
